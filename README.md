# Kubernetes Documentation PDF Generator

This Python tool automatically crawls the Kubernetes official documentation sidebar (`https://kubernetes.io/docs/home/`), extracts every page contained in `<div class="td-content">`, saves each as an individual HTML file, merges related files by alphabetical grouping, and finally converts each merged HTML file into a PDF using Playwright.

All generated PDFs can be imported into **NotebookLM / Gemini** for offline study, search, and summarization.


---

## üì¶ Dependencies

The following libraries and system components must be installed before running the script:

| Dependency | Description | Installation |
|-----------|-------------|---------------|
| Python 3.11+ | Required runtime | https://www.python.org/downloads/ |
| `requests-html` | HTML extraction & session renderer | `pip install requests-html` |
| `playwright` | Used to render and convert HTML to PDF | `pip install playwright` |
| Playwright Chromium | Browser engine for PDF printing | `playwright install chromium` |
| `lxml` | XML/HTML parser backend (often auto-installed via requests-html) | `pip install lxml` |

### Install all dependencies at once

```shell
pip install requests-html playwright lxml
playwright install chromium
```

## How to run the script

### 1. Clone your project

```shell
git clone <your-repo-url>
cd <repository>
```

### 2. Run the script
```shell
python kubernetes-doc.py
```

### 3. Output Directory Structure

After execution, you will see:

```graphql
docs/          # Raw HTML pages extracted from td-content
merged_docs/   # Alphabetically merged grouped HTML files
pdf_docs/      # Final PDFs generated by Playwright
```
